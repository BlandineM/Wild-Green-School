Automatic coffe machine

Stop using unique coffee caspules and get an automatic coffee machine

Objectives

Reduce waste from coffee caspules by using coffee beans and an automatic coffe machine.

Deadlines

This project can be launch immediatly, you just need to buy a coffe machine

Budget

An automatic coffe machine costs between 300€ and 800€. But coffee beans are cheaper than coffe caspsules. Cooffee capsules cost between 20€ and 80€/kg, coffee beans are between 6€ and 15€/kg. Buying an automatic coffee machine is a consequent budget but can be rentabilize quickly

More information

Moreover coffee beans need less industrial modifications, the carbon footprint is by consequence realy lower
